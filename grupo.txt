PROJETO
------------------------------------------------------

Nome: AgroOrb
Slogan: Dados do espaço, colheitas mais inteligentes.
Tema: Indústria Espacial / Agro & Clima

Descrição:
Plataforma web de monitoramento agrícola baseada em dados orbitais,
voltada para pequenos produtores rurais brasileiros. Transforma
imagens e medições de satélites em status visual por setor, alertas
automáticos e recomendações práticas para irrigação, manejo
preventivo e resposta a eventos climáticos extremos.

------------------------------------------------------
COMO RODAR
------------------------------------------------------

Pré-requisito: Node.js >= 20.19

1. Descompacte o arquivo .zip
2. Abra o terminal na pasta do projeto
3. Execute: npm install
4. Execute: npm run dev
5. Acesse http://localhost:5173 no navegador

Para gerar o build de produção: npm run build

------------------------------------------------------
ESTRUTURA
------------------------------------------------------

3 páginas principais navegáveis:
- / (Home) — apresentação do problema e solução
- /painel — dashboard interativo com mapa da fazenda
- /cadastro — cadastro de alertas com validação e persistência

Acessibilidade Libras integrada via VLibras (widget oficial do
governo brasileiro) em todas as páginas.
